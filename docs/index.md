# Multi-Vehicle AV Framework

> A real-time distributed architecture for simulation of autonomous multi-vehicle operations.

- **Distributed** across multiple hosts with Zenoh
- **Autoware + AWSIM Labs** integration
- **Multi-vehicle** orchestration with namespaces
- **AVP** (Automated Valet Parking) demo included

> **Tip:** If you only want AVP, jump to **[AVP → Overview](avp/overview.md)**.

## Why this framework?
- Reproducible multi-host bring-up
- Clean separation between simulation, control, and coordination
- Extensible for ACC, cooperative maneuvers, V2X, and more
