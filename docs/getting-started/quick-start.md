# Quick Start

## Single-Host Minimal Demo (Vehicle 1)
1. Launch **AWSIM Labs**.  
2. Launch **Autoware** (Vehicle 1).  
3. In RViz: **2D Pose Estimate** → **Goal** → **Engage**.  
4. Expected: trajectory appears; vehicle moves in RViz and AWSIM.
